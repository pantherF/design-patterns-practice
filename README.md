# Dive Into Design Patterns — C# Practice Repo

A hands-on C# repository built alongside *Dive Into Design Patterns* by Alexander Shvets (link below). Each section maps to a concept from the book, with small focused projects to put theory into practice.

The exercises were generated by AI (Claude Sonnet), grasping the purpose of the exercises pretty well in my understanding.

---

## About

This repo is a personal learning journal. As I work through the book, I implement the concepts in C# to solidify my understanding — from the fundamentals of OOP all the way through to design patterns. I am already quite familiar with the C# programming language, which is why I picked this one, especially because it implements the OOP concepts well.
